# 🧠 위키자동화 에이전트 (P-Reinforce)
- **핵심 역할**: 파편화된 정보를 영속적 위키로 자동 구조화 및 연결
- **관련 폴더**: `d:\AI City\위키에이전트`
- **설명**: Andre Karpathy의 LLM-Wiki 아키텍처와 강화학습(RL)을 결합하여, 사용자의 원격 데이터를 실시간으로 지식화하고 그래프 연결망을 구축하는 에이전트.

## 📌 주요 기능
- **RL 기반 분류**: 사용자 피드백을 통해 보상 정책($R$)을 최적화하여 분류 정확도 향상.
- **자동 폴더링**: 지식 맥락에 따라 스스로 폴더 트리를 설계하고 확장.
- **GitHub 연동**: 모든 지식 변화를 실시간으로 커밋하여 타임라인 보존.
